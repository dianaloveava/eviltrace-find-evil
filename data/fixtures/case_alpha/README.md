# Case Alpha Fixture

Synthetic demo evidence for EvilTrace. This fixture is intentionally small, safe, and not derived from private data. It exists so judges can run the default demo without paid services, private logs, or external accounts.

Scenario summary: a suspicious login to `workstation-7` is followed by command execution, persistence, archive creation, and outbound upload signals. The exfiltration evidence is intentionally incomplete so EvilTrace can demonstrate self-correction by downgrading an overconfident claim.
